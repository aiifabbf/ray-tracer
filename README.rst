==========
ray-tracer
==========

.. default-role:: math

A toy ray-tracing renderer in Rust

.. figure:: cover.png
    :width: 100%

Quick start
===========

.. code-block:: bash

    cargo run --release > image.png

``--release`` can significantly increase rendering speed.
